## Run Locally

Clone the project

```bash
  git clone https://github.com/Cencent-Proxy/Cencent-Proxy
```

Go to the project directory

```bash
  cd Cencent-Proxy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
